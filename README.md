# نظام تسجيل السفن الإلكتروني - الهيئة العامة للشئون البحرية

نظام متكامل لتسجيل وإدارة السفن إلكترونياً.

## المميزات
- تسجيل سفن جديدة
- متابعة حالة الطلبات
- لوحات تحكم للمسؤولين والموظفين وملاك السفن
- إصدار الشهادات إلكترونياً
- تقارير وإحصائيات متقدمة

## التقنيات المستخدمة
- HTML5, CSS3, JavaScript
- Supabase (قاعدة البيانات والمصادقة)
- Chart.js (الرسوم البيانية)
